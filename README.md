# 📈 Sales & Demand Forecasting (FUTURE_ML_01)

## 🎯 Objective
This project is Task 1 of the **Future Interns** Machine Learning track. The goal is to build a predictive model that forecasts future sales based on historical business data. Accurate demand forecasting helps businesses optimize inventory, manage resources, and improve financial planning.

## 🛠️ Technologies Used
* **Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib
* **Model:** Linear Regression 
* **Environment:** [Jupyter Notebook / VS Code]

## 📊 Project Workflow
1. **Data Generation/Loading:** Created a time-series dataset representing monthly sales over a two-year period.
2. **Data Splitting:** Divided the data into training (80%) and testing (20%) sets.
3. **Model Training:** Trained a Linear Regression model to identify the underlying sales trend.
4. **Evaluation:** Measured model accuracy using Mean Squared Error (MSE) and R-squared ($R^2$) scores.
5. **Forecasting & Visualization:** Predicted sales for the upcoming quarter and visualized the actual vs. predicted trends using Matplotlib.

## 🚀 How to Run the Code
1. Clone this repository: `git clone https://github.com/your-username/FUTURE_ML_01.git`
2. Install the required libraries: `pip install pandas numpy scikit-learn matplotlib`
3. Run the python script: `python task1_sales_forecast.py`

## 🏆 Results
The model successfully identified the upward trend in historical sales and projected realistic targets for the next 3 months. The visualization clearly plots the trajectory, allowing business stakeholders to make informed inventory decisions.
